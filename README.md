# Task Manager (C# Console App)

Простой консольный менеджер задач, написанный на C#.
Проект создан для практики языка C#, работы с коллекциями, LINQ, обработкой ошибок и использованием Git.

---

## 📌 Возможности

* ➕ Добавление задач
* 📋 Просмотр всех задач
* 🔍 Поиск задачи по ID
* ⚠️ Проверка корректности ввода
* 🎨 Цветной вывод сообщений (ошибки, предупреждения, успех)
* 🧰 Вспомогательный класс `ConsoleHelper` для работы с выводом

---

## 🛠 Используемые технологии

* C#
* .NET
* LINQ
* Console API

---

## ▶️ Запуск проекта

1. Клонировать репозиторий

```
git clone https://github.com/ketensaki/TaskManager.git
```

2. Перейти в папку проекта

```
cd MyTaskManager
```

3. Запустить приложение

```
dotnet run
```

Этот проект создан для практики:

* работы с `List`
